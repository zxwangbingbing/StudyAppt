# CC校友录小程序

#### 介绍
CC校友录V10   结合移动互联网时代，独家引入了小程序平台，创造全天候的用户体验。  主要面向毕业后在某城市工作的校友，给大家构建一个充分交流的平台，“人脉”积累是本软件的功能特色，为此淡化了班级，所有的朋友畅所欲言，新老校友信息共享，从不认识到认识，再到熟悉，通过校友身份这个共同的纽带，为自己的人生道路积累持续的人脉。
